# Components Index

| Component | What | When |
|-----------|------|------|
| toast.ts | Shows success/error notifications | Display user feedback after actions |
| form-handler.ts | Handles message submission form validation and upload | User submits new message |
| auth-modal.ts | Sign in/sign up modal with Supabase Auth | User authentication needed |
| plan-modal.ts | Pro upgrade modal with Stripe checkout | User clicks "Go Pro" or reaches free tier limit |
| messages-dashboard.ts | Shows user's messages as cards with pagination (locked/unlocked states) | User signs in (replaces landing page content) |
| message-detail.ts | Modal for locked/unlocked message detail view with countdown or full content | User clicks message card in dashboard |
